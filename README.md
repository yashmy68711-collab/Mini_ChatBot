# Mini Python ChatBot

## About

This is a simple rule-based chatbot built using Python.
The chatbot responds to basic user messages using conditions and keyword matching.

## Features

* Responds to simple greetings
* Keyword-based replies
* Exit option
* Easy command-line interaction

## Tech Used

* Python

## How to Run

1. Run the Python file
2. Type messages to chat with the bot
3. Type `bye` to exit
